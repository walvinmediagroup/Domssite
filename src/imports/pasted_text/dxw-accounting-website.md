Create a premium, modern, fully responsive multi-page website design for **DXW Accounting**, a UK-based remote accounting practice. The website should feel trustworthy, professional, approachable and modern while remaining simple enough for small business owners to immediately understand the services offered.

## Overall Style

Create a dark theme website using charcoal and slate backgrounds with white typography and vibrant orange accents that match the supplied DXW Accounting logo. The orange should be used sparingly for buttons, icons, highlights, hover states and important calls-to-action.

The overall feel should combine the professionalism of a financial institution with the simplicity of a modern SaaS website.

Use:

* Large amounts of whitespace
* Rounded corners (12-16px)
* Soft shadows
* Glassmorphism only where appropriate
* Modern typography (Inter, Manrope or Plus Jakarta Sans)
* Smooth scroll animations
* Premium icons
* Financial illustrations mixed with photography of professionals and UK small businesses
* Subtle animated gradients and orange glow effects

Design everything using Auto Layout, reusable components and variables so the design is production ready.
